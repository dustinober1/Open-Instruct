# Developer Learning Path

This document outlines the skills you will practice and master while building Open-Instruct.

## Level 1: The Foundations (Weeks 1-2)
*Focus: Environment, Basic Python, simple LLM calls.*

- **Skill**: Setting up a Python virtual environment.
- **Skill**: Making a request to a local LLM API (Ollama).
- **Skill**: Defining data structures with Pydantic.
- **Checkpoint**: You can run a script that prints a generated Learning Objective to the console.

## Level 2: The Architect (Weeks 3-4)
*Focus: DSPy, TDD, and Structured Generation.*

- **Skill**: Writing DSPy Signatures (the new "prompt engineering").
- **Skill**: Writing Unit Tests with `pytest` before writing code.
- **Skill**: Debugging LLM hallucinations.
- **Checkpoint**: You have 80%+ success rate in generating valid JSON courses.

## Level 3: The API Designer (Week 4)
*Focus: HTTP, REST, and Error Handling.*

- **Skill**: Building FastAPI endpoints.
- **Skill**: Handling async requests (conceptually).
- **Skill**: Designing proper HTTP 400/500 error responses.
- **Checkpoint**: You can use `curl` or Postman to generate a quiz.

## Level 4: The Full Stack Engineer (Weeks 5-6)
*Focus: Database, Integration, and Frontend.*

- **Skill**: SQL basics (SQLite).
- **Skill**: Connecting a React frontend to a Python backend.
- **Skill**: Managing state (React Flow).
- **Checkpoint**: A fully functional app you can demo to friends.

## Bonus Level: AI Engineer
*Focus: Optimization.*
- **Skill**: Response caching (hashing prompts).
- **Skill**: Switching providers (Strategy Pattern).
- **Skill**: Prompt Optimization (using DSPy optimizers).
